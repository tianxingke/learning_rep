1.mysql解压版设置：
	a.设置PATH ，添加bin的路径
	b.[mysqld] 
		basedir=C:\Program Files\MySQL\MySQL Server 5.6（mysql所在目录） 
		datadir=C:\Program Files\MySQL\MySQL Server 5.6\data （mysql所在目录\data）
	c.以管理员身份运行cd 进入bin文件夹，执行mysqld -install，并执行net start mysql;
	d.设置用户，密码。默认是root但没有密码
		使用mysql -uroot -p 回车后进入，
		use mysql;
		update user set password =password('1234') where user ='root';
		flush privileges; 
		exit ;
	e.设置编码utf-8
		可以自己复制一份.ini文件，命名为my.ini,在sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES 之后添加：
		[client]
		port=3306
		default-character-set=utf8
