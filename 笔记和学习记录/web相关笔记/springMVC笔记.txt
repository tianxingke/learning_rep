3.sessionattributes注解 可以将原本放在请求域中的数据同时也放在session域里边。把数据的名字写进去。可以根据名字，也可以根据类型。此注解只能放在类上边。


@SessionAttributes(value={"user"},types={String.class})  //可以只用一个，也可以同时使用
public XXXclass{

	public String t1(Map<String,Object> map){
		User user = new User();
		map.put("user",user);


		return "";
	}

}
